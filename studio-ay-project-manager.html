🔥 Configurer la synchronisation

Pour synchroniser entre ton ordi et ton téléphone, colle ici ta config Firebase. Comment faire ?

📋 Guide rapide (5 min) :
1. Va sur console.firebase.google.com
2. Clique "Créer un projet" → nom: "hamouni-os" → Continuer
3. Désactive Google Analytics → Créer le projet
4. Clique l'icône </> (Web app) → Enregistre l'app
5. Copie apiKey, projectId, appId ici
6. Dans le menu gauche → Firestore Database → Créer
7. Choisis "Mode test" → Activer
C'est tout ! 🎉
